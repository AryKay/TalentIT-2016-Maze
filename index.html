TalentIT 2016 Challenge


Welcome to Smarp's mini challenge! Try solving it and receive a reward from us at the TalentIT 2016 fair.

Below is an image of a maze. You need to get from the START point to the END point in only 20 moves. You have two sets of tools to solve the challenge:

  • Single-move: You can move one step at a time using this tool. Keep track of your moves though, as each step counts as one move!
  • While-loop Constructor: You can create while loops depending on a single condition and travel longer distances while saving moves!
